# OpenLang
<br>
##Goal:
OpenLang is an open-source dataset containing linguistic data from multiple languages. Our primary goal is to collect and preserve data on endangered languages. Another goal of OpenLang is to provide valuable linguistic resources to support the development of AI models with multilingual capabilities. We hope this project helps the survival of endangered languages and helps AI models lern theese languages aswell.
